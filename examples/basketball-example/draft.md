# Stat read: Demo Guard 2026 — usage with efficiency

On a 62-game synthetic regular-season sample, Demo Guard carried a **27.8% usage rate** as the demo team's primary on-ball guard. The important number isn't the usage alone — it's **59.2% true shooting** on that volume. That's the difference between "high-usage star" as label and high-usage star as **efficient** offensive load.

True shooting here is doing the work usage rate can't: it prices in how costly those possessions are. If the efficiency were merely average for that usage band, he'd be a classic **usage-illusion** case — lots of touches, debatable offensive value. The current read is the opposite: the model should like him when it weights scoring efficiency against involvement.

A stat system can fairly stay skeptical if its formula heavily weights playmaking defense or playoff matchup hunting — those are open questions here, not contradictions.

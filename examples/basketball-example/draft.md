# Stat read: Brunson 2024-25 — usage with efficiency

On a 68-game regular-season sample, Brunson carried a **28.4% usage rate** as New York's primary on-ball guard. The important number isn't the usage alone — it's **58.1% true shooting** on that volume. That's the difference between "high-usage star" as label and high-usage star as **efficient** offensive load.

True shooting here is doing the work usage rate can't: it prices in how costly those possessions are. If the efficiency were merely average for that usage band, he'd be a classic **usage-illusion** case — lots of touches, debatable offensive value. The current read is the opposite: the model should like him when it weights scoring efficiency against involvement.

What TS% still misses in this packet: **assist/turnover detail is not fully loaded** (AST% marked unknown), and there's **no defensive characterization**. A stat system can fairly stay skeptical if its formula heavily weights playmaking defense or playoff matchup hunting — those are open questions here, not contradictions.

Bottom line for 2024-25: the stat story is "primary initiator who doesn't waste possessions at scale." Skepticism should target what isn't in the box score yet, not the efficiency-on-usage core.

# v0.1.0 DOI-Ready Release Notes

## Archived In This Release

- RDW CLI source code, validators, domain assets, prompt assets, and installer
  templates.
- Curated examples for basketball, music, technical, and batch planning.
- Limitations and integration documentation.
- Citation, Zenodo, and research-readiness metadata.

## Excluded From The Archive

- User-specific generated `.rdw-runs/` outputs.
- Private source packets or unpublished research materials.
- Host-specific installed skill copies outside the repository.

## Validation Path

```bash
uv run rdw doctor
uv run rdw validate-batch examples/batch-tasks.yaml
uv run pytest
uv run ruff check src tests
uv run basedpyright src tests
```

## Supported Claims

- RDW validates structured research packets and batch files.
- RDW emits repeatable task contracts and prompt bundles for agents to execute.
- RDW does not browse, call model APIs, or draft autonomously in v0.1.

## Required Caveats

- Output quality depends on the agent and source material supplied to the run.
- Example domains demonstrate schema and claim-boundary behavior, not universal
  writing quality.
- Generated run artifacts should be cited only when their source packets are
  included or otherwise available.

## DOI Minting Checklist

- Confirm author ORCID in citation metadata.
- Confirm examples contain no private source material.
- Run the validation path on the release commit.
- Tag the DOI-ready release as `v0.1.0` or a later explicit release version.

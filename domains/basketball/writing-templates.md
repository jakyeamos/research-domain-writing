# Basketball Writing Templates

## player_summary

3–6 sentences: role → two evidence-backed traits → one limit → confidence note if low.

## stat_interpretation

1. Metric + value + sample  
2. What it measures  
3. What it misses for this player  
4. Role-sensitive read  
5. Caveat

## why_stat_likes_player

Mechanism chain: stat input → player skill/role → why not noise.

## why_stat_skeptical

Mismatch: what the stat penalizes that role/team still needs, or efficiency cost.

## ranking_explanation

Leaderboard/surface copy: what the ranking measures → how to read order → 1–2 caveats (sample, role, stat limits). Row blurbs optional in same task or batch. Do not invent metric definition — must be in research packet.

## comparison_blurb

Shared role context → 2–3 dimensions with evidence → explicit non-claim.

## sleeper_case / overrated_warning

Claim → stat-role evidence → what would falsify the take.

## future_projection

Ceiling/floor drivers → dependency (health, shooting variance, team fit) → uncertainty band.

## team_fit_notes

Scheme fit (spacing, PNR, switching) → roster redundancy → on/off fit if in packet.
